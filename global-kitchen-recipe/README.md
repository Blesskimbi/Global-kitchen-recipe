# the global gitchen API KIMBI BLESS TANGIRI

A RESTful API for managing a digital cookbook.

## Features
- Create, read, update, delete recipes
- Filter recipes by category
- Layered architecture (Routes, Controllers, Services, Models)
- MongoDB schema validation and indexing
- Global error handling


## API Endpoints
- GET `/recipes` (optional `?category=...`)
- POST `/recipes`
- GET `/recipes/:id`
- PATCH `/recipes/:id`
- DELETE `/recipes/:id`

## Tech Stack
- Node.js
- Express.js
- MongoDB with Mongoose
- dotenv
